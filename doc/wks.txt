1. Intro                                                       *wks-intro*

wks.nvim is a thin launcher for the external |_wks| TUI.
It does not manage workspace data directly in Neovim.

==============================================================================
2. Requirements                                                 *wks-reqs*

- Neovim 0.11+
- The |_wks| executable available on PATH, or configured explicitly

==============================================================================
3. Setup                                                       *wks-setup*

The plugin in this repository lives under `neovim/`, so URL-based installs
must prepend that subdirectory to runtimepath.

>
  {
    url = "https://github.com/callmiy/workspace-manager.git",
    name = "wks.nvim",
    init = function(plugin)
      vim.opt.rtp:prepend(plugin.dir .. "/neovim")
    end,
    build = function(plugin)
      vim.cmd("helptags " .. vim.fn.fnameescape(plugin.dir .. "/neovim/doc"))
    end,
    config = function()
      require("wks").setup({
        binary = "_wks",
        default_mode = "tab",
        float = {
          width = 0.9,
          height = 0.9,
          border = "rounded",
        },
      })
    end,
  }
<

==============================================================================
4. Commands                                                    *wks-cmds*

                                                        *:Wks*
:Wks
    Open |_wks| with configured default mode.

                                                        *:Wks-tab*
:Wks tab
    Open |_wks| in a new tab terminal.

                                                        *:Wks-float*
:Wks float
    Open |_wks| in a floating terminal.

==============================================================================
5. Configuration                                              *wks-config*

                                                        *wks.setup()*
require("wks").setup(opts)

Options:

                                                        *wks-config-binary*
binary
    string, default "_wks"
    Executable name or absolute path.

                                                  *wks-config-default_mode*
default_mode
    "tab" | "float", default "tab"

                                                        *wks-config-float*
float
    table for floating window options:
    - width: number, default 0.9
    - height: number, default 0.9
    - border: string, default "rounded"
